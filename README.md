Websocket tutorial
===

> javascript, python

Get source
---

```
git clone https://github.com/kmkc/websocketTutorial.git
```

Change souce
---

You should change `hogehoge.xyz:3000` in client.py

Start demo
---

0.  Start server: `python server.py`
0.  Web access to `hogehoge.xyz:3000`
0.  Run `python client.py`
